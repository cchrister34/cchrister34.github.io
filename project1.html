About the project

Cosmos Conqueror was developed as part of my coursework for my games programming module during my second year at Northumbria University. It was my first game made in C++ inside a primitive engine built around the SDL library provided by the university. Because of the limitations of the engine, I was working at a lower level of abstraction than commercial engines like Unreal Engine, which provided me with a deeper understanding of the fundamentals of game development.

Collision System

The Process Collision function uses the GameObject superclass to check for collisions between game objects using if statements to check the type of object that the spaceship has collided with and then execute the relevant logic. The engine also included an event system that allowed for more complex interactions between objects by setting the event from an "Event Type" enum that is then accessed in the game manager's Handle Event function to execute the relevant logic for the event.

What I learned

This project was my first experience working with C++ in a game development context, and it provided me with a solid foundation in the language and its application in game development. Working with a primitive engine built around SDL allowed me to understand the lower-level workings of game engines, including rendering, input handling, and game loop management. I also gained experience in object-oriented programming (OOP) principles.

One of the key challenges I faced during this project was coupling as before creating the project, i did not properly plan out how game objects would interact with each other, which led to tight coupling between classes. This made it diffiucult to make changes as my project progressed, specifically to the spaceship class which became monolithic with the amount of logic implemented.

Overall, this project was a valuable learning experience that helped me develop my skills in C++ programming and game development. It showed me the importance of planning and designing by utilsing inheritance and polymorphism to create a more flexible and maintainable codebase.
