Cosmos Conqueror

Project Deep Dive

Cosmos Conqueror was developed as part of my coursework for my games programming module during my second year at Northumbria University. It was my first game made in C++ inside a primitive engine built around the SDL library provided by the university. Because of the limitations of the engine, I was working at a lower level of abstraction than commercial engines like Unreal Engine, which provided me with a deeper understanding of the fundamentals of game development.

Collision Detection System C++

Collision Detection Logic

This function implements circular collision detection by calculating the distance between two objects and comparing it to the sum of their radii. Essential for detecting when the player's ship collides with asteroids or power-ups in the game world.

Spaceship Physics C++

Realistic Movement System

The spaceship's physics system applies thrust-based acceleration using trigonometry, implements friction for momentum decay, and handles screen wrapping. This creates the classic Asteroids-style movement feel with smooth, realistic physics.
